This is the resume template that I have maintained since college, updated when bugs or other issues are discovered.

Check out automatically generated examples on GitHub Pages!

Template Source: github.com/MTecknology/latex-resume

Goals: Readable, Maintainable, & ATS-Friendly

Features:
- Includes examples of CV.tex and Resume.tex
- Optimized for Applicant Tracking Systems (ATS)
  * validated using Open Resume Parser,
  * white space, no smallcaps, text output
  * single column, uppercase sections, etc.
- Option to toggle redaction of private information
- Option to create customized page numbering
- Avoids excessive tweaks, letting LaTeX shine
- Also avoids magic hacks and complicated usage
- Creates document outline and PDF bookmarks
- Documents all available features in mteck.sty
- Uses as much page space as possible
- Ensures page can be printed within safe margins
- Keeps content separate from style
- Includes careful content alignment
- Style prefers "em" to support flexible font size
- Example includes "standard" social link blocks
- Resume format includes "Breakthrough!" advice
  * from Hudson Valley Career Development
- Puts emphasis on the applicant, not employers
- Continually improved (report an issue)
